<task type="auto">
  <name>Task 3: Add GSAP with Svelte action pattern</name>
  <files>
    package.json
    src/lib/actions/animate.ts
    src/routes/+page.svelte
  </files>
  <action>
    1. Install GSAP:
       `npm install gsap`

    2. Create src/lib/actions/animate.ts with cleanup pattern:
       ```typescript
       import { gsap } from 'gsap';

       export function animate(
         node: HTMLElement,
         params: gsap.TweenVars
       ): { destroy: () => void } {
         const tween = gsap.from(node, params);

         return {
           destroy() {
             tween.kill();
           }
         };
       }
       ```

    3. Update src/routes/+page.svelte to test GSAP (must use onMount to avoid SSR):
       ```svelte
       <script>
         import { onMount } from 'svelte';

         let gsapLoaded = $state(false);

         onMount(async () => {
           const { gsap } = await import('gsap');
           gsapLoaded = true;

           gsap.from('.test-box', {
             opacity: 0,
             y: 20,
             duration: 0.5
           });
         });
       </script>

       <main class="min-h-screen bg-nbrs-green p-8">
         <h1 class="text-4xl font-bold text-white mb-4">NBRS Website v4</h1>

         <div class="space-y-2 text-white">
           <p>Tailwind: <span class="font-bold text-nbrs-coral">Working</span></p>
           <p>GSAP: <span class="font-bold">{gsapLoaded ? 'Loaded' : 'Loading...'}</span></p>
         </div>

         <div class="test-box mt-8 p-4 bg-white rounded-lg text-nbrs-green font-bold">
           If you see this animated in, GSAP works!
         </div>
       </main>
       ```

    IMPORTANT: GSAP must be dynamically imported in onMount to avoid SSR errors.
    The action in animate.ts is for use with `use:animate` directive in future components.
  </action>
